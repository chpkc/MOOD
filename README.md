# MOOD (Moodify)

Музыкальный дашборд на React, использующий Spotify API для отображения вашей статистики.

## Возможности

- **Топ треков**: Просмотр самых прослушиваемых треков за 4 недели, 6 месяцев и всё время.
- **Топ артистов**: Ваши любимые исполнители.
- **Настроение**: Анализ музыкальных предпочтений (энергия, танцевальность, позитивность) с визуализацией на радаре.
- **История**: Список недавно прослушанных треков.

## Установка и запуск

1. **Клонируйте репозиторий** (или скачайте код).

2. **Установите зависимости**:
   ```bash
   npm install
   ```

3. **Создайте приложение в Spotify Developer Dashboard**:
   - Перейдите на [Spotify Developer Dashboard](https://developer.spotify.com/dashboard).
   - Нажмите "Create App".
   - Заполните название и описание.
   - В настройках приложения добавьте **Redirect URI**:
     - `http://localhost:5173/callback`
     - `http://127.0.0.1:5173/callback`

4. **Настройте переменные окружения**:
   - Создайте файл `.env` в корне проекта.
   - Добавьте ваш Client ID:
     ```
     VITE_SPOTIFY_CLIENT_ID=ваш_client_id
     ```

5. **Запустите проект**:
   ```bash
   npm run dev
   ```

## Стек технологий

- React 18 + Vite
- Tailwind CSS
- React Router v6
- Recharts
- Spotify Web API (OAuth 2.0 PKCE)
